## Overview
A powerful, interactive Streamlit application built with Python and SymPy to perform indefinite symbolic integration. It provides step-by-step mathematical reasoning, method detection (U-Substitution, Partial Fractions, Basic Patterns), runtime graphing, and verification auditing.

## Setup & Running Instructions (using `uv`)
1. Ensure you have `uv` installed.
2. Clone this repository and navigate to the project folder.
3. Install dependencies using uv:
   ```bash
   uv pip install streamlit sympy numpy plotly
   # OR
   uv sync
4. Run the application:
    uv run streamlit run app.py
